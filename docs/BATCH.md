● マッパー内で分岐に使える情報一覧

  マッパーのシグネチャは (item: LineItem, ctx: MapperContext) -> str | None です。

  ---
  1. item: LineItem から得られる情報

  ┌───────────────┬───────────────────────────┬───────────────────────────────────────────────────────┐
  │  フィールド   │            型             │                         意味                          │
  ├───────────────┼───────────────────────────┼───────────────────────────────────────────────────────┤
  │ concept       │ str                       │ Clark記法のQName ({namespace}localName)               │
  ├───────────────┼───────────────────────────┼───────────────────────────────────────────────────────┤
  │ namespace_uri │ str                       │ 名前空間URI。標準タクソノミか提出者別かの判別に使える │
  ├───────────────┼───────────────────────────┼───────────────────────────────────────────────────────┤
  │ local_name    │ str                       │ 要素のローカル名 (例: NetSales)                       │
  ├───────────────┼───────────────────────────┼───────────────────────────────────────────────────────┤
  │ label_ja      │ LabelInfo                 │ 日本語ラベル（.text, .role, .source）                 │
  ├───────────────┼───────────────────────────┼───────────────────────────────────────────────────────┤
  │ label_en      │ LabelInfo                 │ 英語ラベル                                            │
  ├───────────────┼───────────────────────────┼───────────────────────────────────────────────────────┤
  │ value         │ Decimal | str | None      │ 実際の値                                              │
  ├───────────────┼───────────────────────────┼───────────────────────────────────────────────────────┤
  │ unit_ref      │ str | None                │ 単位参照 (例: JPY)                                    │
  ├───────────────┼───────────────────────────┼───────────────────────────────────────────────────────┤
  │ decimals      │ int | "INF" | None        │ 精度                                                  │
  ├───────────────┼───────────────────────────┼───────────────────────────────────────────────────────┤
  │ context_id    │ str                       │ コンテキストID                                        │
  ├───────────────┼───────────────────────────┼───────────────────────────────────────────────────────┤
  │ period        │ InstantPeriod |           │ 期間（instant=BS系, duration=PL/SS/CF系）             │
  │               │ DurationPeriod            │                                                       │
  ├───────────────┼───────────────────────────┼───────────────────────────────────────────────────────┤
  │ entity_id     │ str                       │ エンティティ識別子                                    │
  ├───────────────┼───────────────────────────┼───────────────────────────────────────────────────────┤
  │ dimensions    │ tuple[DimensionMember,    │ ディメンション情報（軸+メンバー）                     │
  │               │ ...]                      │                                                       │
  ├───────────────┼───────────────────────────┼───────────────────────────────────────────────────────┤
  │ is_nil        │ bool                      │ xsi:nil=true かどうか                                 │
  ├───────────────┼───────────────────────────┼───────────────────────────────────────────────────────┤
  │ order         │ int                       │ 文書内出現順                                          │
  └───────────────┴───────────────────────────┴───────────────────────────────────────────────────────┘

  LabelInfo.source は STANDARD（標準タクソノミ由来）/ FILER（提出者別）/ FALLBACK の3値。

  ---
  2. ctx: MapperContext から得られる情報

  ┌─────────────────────────┬────────────────────────────┬─────────────────────────────────┐
  │       フィールド        │             型             │              意味               │
  ├─────────────────────────┼────────────────────────────┼─────────────────────────────────┤
  │ dei                     │ DEI | None                 │ 文書・企業情報（下記参照）      │
  ├─────────────────────────┼────────────────────────────┼─────────────────────────────────┤
  │ detected_standard       │ DetectedStandard | None    │ 検出された会計基準              │
  ├─────────────────────────┼────────────────────────────┼─────────────────────────────────┤
  │ industry_code           │ str | None                 │ 業種コード (例: "bk1" = 銀行業) │
  ├─────────────────────────┼────────────────────────────┼─────────────────────────────────┤
  │ definition_parent_index │ dict[str, str]             │ 提出者概念→標準祖先の逆引き辞書 │
  ├─────────────────────────┼────────────────────────────┼─────────────────────────────────┤
  │ calculation_linkbase    │ CalculationLinkbase | None │ 計算リンクベース                │
  └─────────────────────────┴────────────────────────────┴─────────────────────────────────┘

  DEI の主要フィールド

  - accounting_standards — 会計基準 (J-GAAP, IFRS, US-GAAP, JMIS)
  - has_consolidated — 連結の有無
  - industry_code_consolidated / industry_code_non_consolidated — 業種コード
  - type_of_current_period — "FY"(通期) or "HY"(半期)
  - current_fiscal_year_start_date / current_period_end_date — 会計期間の日付群
  - security_code / edinet_code / fund_code — 企業識別情報

  DetectedStandard のフィールド

  - standard — 会計基準 (J-GAAP等)
  - method — 検出方法 (DEI / NAMESPACE / UNKNOWN)
  - detail_level — DETAILED(詳細) or BLOCK_ONLY(ブロックのみ)
  - has_consolidated — 連結有無
  - period_type — "FY" or "HY"
  - namespace_modules — 使用名前空間モジュール集合 ({"jppfs", "jpcrp", ...})

  CalculationLinkbase のメソッド

  - ancestors_of(concept, role_uri=...) — 計算木での祖先概念を取得（近→遠順）
  - children_of(parent, role_uri=...) — 子ノードの CalculationArc を取得

  ---
  3. 現在のマッパーが実際に分岐に使っている情報

  ┌─────────────────────────┬───────────────────────────────────────────────────────────────────┐
  │        マッパー         │                           主に使う情報                            │
  ├─────────────────────────┼───────────────────────────────────────────────────────────────────┤
  │ summary_mapper          │ item.local_name の辞書引き                                        │
  ├─────────────────────────┼───────────────────────────────────────────────────────────────────┤
  │ statement_mapper        │ item.local_name → 完全一致 → 正規化一致（IFRS等サフィックス除去） │
  ├─────────────────────────┼───────────────────────────────────────────────────────────────────┤
  │ standard_concept_mapper │ item.namespace_uri が標準タクソノミかどうか                       │
  ├─────────────────────────┼───────────────────────────────────────────────────────────────────┤
  │ definition_mapper       │ ctx.definition_parent_index[item.local_name] で祖先を引く         │
  ├─────────────────────────┼───────────────────────────────────────────────────────────────────┤
  │ calc_mapper             │ ctx.calculation_linkbase.ancestors_of() で計算木を辿る            │
  └─────────────────────────┴───────────────────────────────────────────────────────────────────┘

  ---
  まとめ

  分岐に使える主な軸は:

  1. 概念名 (local_name) — 辞書引き
  2. 名前空間 (namespace_uri) — 標準 vs 提出者別
  3. 会計基準 (ctx.detected_standard.standard) — J-GAAP/IFRS/US-GAAP/JMIS
  4. 業種 (ctx.industry_code) — 銀行・建設・鉄道等
  5. 期間種別 (period_type) — 通期/半期
  6. ディメンション (item.dimensions) — 連結/個別、セグメント等
  7. 定義リンクベース (definition_parent_index) — 提出者概念の標準祖先
  8. 計算リンクベース (calculation_linkbase) — 計算木の親子関係
  9. ラベル (label_ja.text) — 日本語名での判別
