  まとめると

  jpxbrl (共通コア) に入るもの:
  ├── parser.py          ← edinet.xbrl.parser をそのまま
  ├── contexts.py        ← edinet.xbrl.contexts をそのまま
  ├── units.py           ← edinet.xbrl.units をそのまま
  ├── linkbase/          ← edinet.xbrl.linkbase をそのまま
  ├── _linkbase_utils.py ← そのまま
  ├── _namespaces.py     ← XBRL標準定数のみ抽出
  └── exceptions.py      ← ParseError 等の共通例外

  edinet に残るもの:
  ├── api/               ← EDINET API 固有
  ├── models/            ← Filing, Company, DocType 等
  ├── xbrl/taxonomy/     ← EDINET タクソノミ解決
  ├── xbrl/dei.py        ← EDINET の DEI
  ├── xbrl/facts.py      ← LineItem 構築(タクソノミ依存)
  ├── financial/         ← 財務諸表組立・名寄せ
  ├── display/           ← 表示
  └── dataframe/         ← エクスポート

  tdnet (新規):
  ├── api/               ← TDnet ファイル取得
  ├── models/            ← 短信モデル
  ├── taxonomy/          ← TSE タクソノミ解決
  └── financial/         ← 短信の構造化
