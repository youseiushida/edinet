## C-12. XBRL Formula / Assertion Linkbase

### 質問への対応

| サブ質問 | 回答 |
|----------|------|
| C-12.1 Formula/Assertion LB の有無 | EDINET タクソノミには Formula Linkbase / Assertion Linkbase は含まれていない |
| C-12.2 パーサーでの処理要否 | 不要。計算リンクベース（summation-item）のみで計算検証は完結する |
| C-12.3 Arelle 等外部ツールへの委譲 | 委譲の必要なし。Formula/Assertion LB が存在しないため処理対象がない |

### 情報源（Fact）

- [F1] スクリプト: `docs/QAs/scripts/C-12.formula_search.py` 実行結果 — タクソノミ全体（2690 件の XML/XSD ファイル）を走査し、ファイル名に "formula"/"assertion"/"variable" を含むファイルは 0 件、ファイル内容に `xbrl.org/2008/formula`、`xbrl.org/2008/assertion`、`xbrl.org/2008/variable` 等の Formula 関連 namespace を含むファイルも 0 件
- [F2] パス: `docs/仕様書/EDINETタクソノミの設定規約書.md` L.1648-1650 — 「EDINET タクソノミの関係層に当たる各種ファイル名の命名規約について『図表 3-3-1 表示リンクベースファイル名』、『図表 3-3-2 定義リンクベースファイル名』及び『図表 3-3-3 計算リンクベースファイル名』に示す。」（関係層は表示・定義・計算の 3 種のみと明記されており、Formula LB への言及なし）
- [F3] パス: `docs/仕様書/EDINETフレームワーク設計書.md` L.794 — 「関係層を構成する関係リンクベースファイル（表示リンク、定義リンク及び計算リンク）、パターン別関係リンクベースファイルで定義する内容は、次のとおりである。」（同様に Formula LB への言及なし）
- [F4] 仕様書全文検索 — `docs/仕様書/` 配下の全仕様書（設定規約書、概要説明、フレームワーク設計書、提出者別タクソノミ作成ガイドライン、提出書類ファイル仕様書、XBRL Instance Guideline、および 2026 年版仕様書）について "Formula"/"Assertion" で Grep 検索した結果、該当なし

### 推論（Reasoning）

1. [F1] より、ALL_20251101 タクソノミの 2690 件の XML/XSD ファイルにおいて、Formula/Assertion/Variable に関連するファイル名・namespace・arcrole は一切存在しない。
2. [F2] + [F3] より、EDINET の仕様書は関係層のリンクベースを「表示リンク」「定義リンク」「計算リンク」の 3 種類のみと定義しており、Formula Linkbase は設計上含まれていない。
3. [F4] より、全仕様書において "Formula" や "Assertion" という用語への言及が皆無であり、EDINET が XBRL Formula 仕様を採用していないことが確認できる。
4. [F1] + [F2] + [F3] + [F4] より、EDINET タクソノミには Formula/Assertion Linkbase は存在せず、パーサーでの処理は不要と結論する。計算検証が必要な場合は Calculation Linkbase（summation-item arcrole）のみを処理すればよい。

### 確信度

- 高（タクソノミ全ファイルの実走査 + 全仕様書の全文検索の両方で該当なし）
