## E-4. EPS・BPS 等の 1 株当たり情報

### 質問への対応

q.md のサブ質問ごとに回答箇所を対応付ける。

| サブ質問 | 回答 |
|----------|------|
| E-4.1 concept の所属名前空間 | **`jpcrp_cor`**（開示府令語彙）に属する。`jppfs_cor`（財務諸表語彙）ではない。EPS/BPS は経営指標等（連結経営指標等・提出会社の経営指標等）のセクションに含まれるため、財務諸表本表の名前空間ではなく開示府令の名前空間で定義されている。 |
| E-4.2 unit の表現 | **`xbrli:divide`** で表現される。unit ID は `JPYPerShares`。分子は `iso4217:JPY`、分母は `xbrli:shares`。データ型は `num:perShareItemType`。 |
| E-4.3 潜在株式調整後 EPS | **別の concept が存在する**。日本基準: `DilutedEarningsPerShareSummaryOfBusinessResults`。IFRS/JMIS/US-GAAP 版もそれぞれ独立した concept が存在する。 |

### Concept の詳細

全ての EPS/BPS concept は `jpcrp_cor_2025-11-01.xsd` に定義されている。`jppfs_cor` には 1 株当たり情報の concept は存在しない。

#### 基本的な 1 株当たり情報（日本基準）

| concept 名 | 日本語ラベル | データ型 | periodType |
|------------|-------------|---------|------------|
| `NetAssetsPerShareSummaryOfBusinessResults` | 1 株当たり純資産額（BPS） | `num:perShareItemType` | instant |
| `BasicEarningsLossPerShareSummaryOfBusinessResults` | 1 株当たり当期純利益又は当期純損失（EPS） | `num:perShareItemType` | duration |
| `DilutedEarningsPerShareSummaryOfBusinessResults` | 潜在株式調整後 1 株当たり当期純利益 | `num:perShareItemType` | duration |

- BPS は `periodType="instant"`（期末時点の値）
- EPS は `periodType="duration"`（会計期間の値）
- XSD 定義行: BPS = L.828、基本 EPS = L.829、希薄化後 EPS = L.830

### Unit の定義

1 株当たり金額の単位は `xbrli:divide` 要素で定義される。

```xml
<xbrli:unit id="JPYPerShares">
  <xbrli:divide>
    <xbrli:unitNumerator>
      <xbrli:measure>iso4217:JPY</xbrli:measure>
    </xbrli:unitNumerator>
    <xbrli:unitDenominator>
      <xbrli:measure>xbrli:shares</xbrli:measure>
    </xbrli:unitDenominator>
  </xbrli:divide>
</xbrli:unit>
```

ガイドライン 図表 5-5-5 に「1 株当たりの金額」の単位仕様が定義されている:

| 項目 | 値 |
|------|-----|
| unit ID | `JPYPerShares` |
| データ型 | `perShareItemType` |
| measure | `iso4217:JPY / xbrli:shares` |
| 用途 | 1 株当たりの金額（日本円）に利用 |

関連する基本単位:
```xml
<xbrli:unit id="JPY">
  <xbrli:measure>iso4217:JPY</xbrli:measure>
</xbrli:unit>

<xbrli:unit id="shares">
  <xbrli:measure>xbrli:shares</xbrli:measure>
</xbrli:unit>
```

### 推論（Reasoning）

1. **E-4.1 concept の所属について**: [F1]-[F7] より、EPS/BPS 関連の全 concept は `jpcrp_cor_2025-11-01.xsd` に定義されている。[F8] より、`jppfs_cor` には 1 株当たり情報の concept は存在しない。これは、EPS/BPS が財務諸表本表（BS/PL/CF/SS）の科目ではなく、有価証券報告書の「主要な経営指標等の推移」セクション（開示府令のフォーマット）に含まれるデータであるためと理解できる。パーサー実装では `jpcrp_cor` 名前空間の concept として扱う必要がある。

2. **E-4.2 unit の表現について**: [F9][F10] より、1 株当たり金額の単位は `xbrli:divide` で「円/株」を表現する。単位 ID は `JPYPerShares` で固定。[F11] のサンプルインスタンスでも同一の定義を確認。通貨単位 + `decimals` による処理ではなく、明示的な除算単位が使われる。データ型は `num:perShareItemType` であり、これは XBRL International が定義する数値型。

3. **E-4.3 潜在株式調整後 EPS について**: [F3] より、日本基準の希薄化後 EPS は `DilutedEarningsPerShareSummaryOfBusinessResults` として独立した concept が存在する。[F4]-[F6] より、IFRS/JMIS/US-GAAP 各版にもそれぞれ独立した希薄化後 EPS concept が存在する。命名パターンは `Diluted` プレフィックス + 会計基準サフィックス。[F12] のサンプルインスタンスでも基本 EPS と希薄化後 EPS が別の Fact として記載されている。
